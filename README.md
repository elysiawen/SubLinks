# Vercel Subscription Manager

Based on Next.js 14+ and Vercel KV.

## Deployment

1. Deploy to Vercel.
2. Create Vercel KV Database (Storage tab).
3. Set `ADMIN_PASSWORD` in Environment Variables.

## Features

- **Admin Panel**: `/admin`
- **Subscription**: `/api/s/[token]`
- **UA Blocking**: WeChat/QQ blocked.
- **Caching**: 24h default.

See `walkthrough.md` in the artifacts for full details.
